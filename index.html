Imagen responsiva

Para que una imagen sea responsiva se le debe asignar la clase “img-responsive”, con esta clase la imagen se adaptara según la resolución actual.
Ejemplo:


<img src="Ruta imagen" alt="imagen" class="img-responsive">

Si quieres establecer un tamaño para la imagen responsiva se le debe agregar la clase img-width-x donde la x es un valor del 1 al 100.
Ejemplo:


<img src="Ruta imagen" alt="imagen" class="img-responsive img-width-50">

Agregar enlace.

Enlace marcado

Para agregar un enlace con estilo marcado, se necesita utilizar la clase mark__green en una etiqueta <a>.
Ejemplo:


<p><a class="mark__green" href="#">texto </p>

Las clases disponibles son:
mark__green, mark__orange, mark__blue, mark__pink, mark__purple, mark__yellow, mark__red y mark__gray, cada una con un color específico.

Enlace boton

Para agregar un enlace con estilo de botón, se necesita utilizar la clase link__button__green en una etiqueta <a>.
Ejemplo:


<p<a class="link__button__green" href="#">texto </a></p>


Las clases disponibles son:
link__button__green, link__button__orange, link__button__blue, link__button__pink, link__button__violet, link__button__yellow, link__button__red y link__button__gray, cada uno con un color específico.

Enlace imagen

Para agregar un enlace en una imagen, se necesita utilizar la clase image__link en una etiqueta <img> que esta adentro de una etiqueta <a>.
Ejemplo:


<a href="#"> <img src = "ruta imagen" alt = "descripción de la imagen" class = "image__link" > </a>
